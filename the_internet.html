Clients and Servers

Computers connected to the Web are called clients and servers.

  • Clients are the typical Web user's Internet-connected devices (for example, your computer connected to your Wi-Fi, or your phone connected to your mobile network) and Web-accessing software available on those devices.
  • Servers are computers that store webpages webpages available on those devices (usually a web-browser such as Firefox or Chrome.

So What Actually Happens?

  1. The browser goes to the DNS Server and finds the real address value of the server that the website lives on.
  2. The browser sends an HTTP HTTP Request message to the server asking it to send a copy of the website to the client.
  3. If the server approves the client's request, the server sends the client a "200 OK message" message to the server asking it to send a copy of the website to the client.
  4. The browser assembles the small chunks into a complete website and displays it to you!
